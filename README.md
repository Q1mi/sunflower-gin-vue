# sunflower
一个 Go Web 实战项目，基于 gin + GORM + Vue3 + Vite 开发的签到中心系统。
非常适合新手入门学习，从零开始搭建一个完整的 Web 项目。

## 目录结构

```
├─api.json                  // 接口文档
├─sunflower-gin             // 后端代码
├─frontend                  // 前端代码
└─README.md                 // 项目说明文件
```

## 技术栈

### 后端（backend）
- Gin： 社区中最流行的 Web 框架。
- Redis：用于缓存。
- MySQL：用于存储数据。
- Snowflake：用于生成全局唯一ID。

### 前端（frontend）
- Vue3：最新的Vue版本，提供了更强大的功能和更好的性能。
- Vite：下一代前端开发与构建工具，支持热更新等功能。
